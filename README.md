This is a Web Terminal application.

The tach stack is:
 - Node
 - Express 4.2
 - Socket.IO
 - Socket.IO-stream
 - Dockerode
 - Docker
 - AngularJS

To run it, you have to do:

STEP 1:

```
npm install
bower install
```

STEP 2:

install docker and pull some images.

STEP 3:

```
node app.js
```

STEP 4:

visit http://localhost:3000 and select one image to start a container.
